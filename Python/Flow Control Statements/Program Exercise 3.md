# Check whether a number is Prime or not

- A prime number is a whole number greater than 1 that has only two factors: 1 and the number itself.

- Any whole number greater than 1 that is not a prime number is a composite number. This means it has more than two factors.

![[/image 52.png|image 52.png]]

```python
n = int(input("Enter a number: "))
res = "Prime"
for i in range(2, n):
    if n % i == 0:
       res = "Non-Prime"
       break
print(f"{n} is {res}")
```

> Output
> 
> Enter a number: 2  
> 2 is Prime
> 
> Enter a number: 4  
> 4 is Non-Prime

# Tom and Jerry (Odd or Even)

Computer generates a random number between 0 to 6

User enters a number n. Add n to users score if it’s not equal to the number generated by the computer.

When user fails (n is equal to the computer’s number) show him his score.

link : [https://www.geeksforgeeks.org/python/random-numbers-in-python/](https://www.geeksforgeeks.org/python/random-numbers-in-python/) to generate random number

```python
import random
score = 0
while True:
    rand = random.randint(0, 6)
    num = int(input("Enter your number: "))
    print(f"Random number: {rand}")
    if rand == num:
        break
    if num == 0:
        score += rand
    score += num
    print(f"Score is: {score}")
print(f"You failed! your score is {score}")
```

> Output
> 
> Enter a number : 0  
> Random number: 6  
> You scored : 6  
> Enter a number : 1  
> Random number: 4  
> You scored : 7  
> Enter a number : 4  
> Random number: 4  
> You failed! Your score is : 7
