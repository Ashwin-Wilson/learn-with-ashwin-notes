Let’s get started by understanding some basic of Computer.

## Input and Output Devices

- **Input Device**: Devices that are used to send information to the computer.
    
    eg: Keyboard, Mouse, Touchpad, Microphone, etc
    

- **Output Device**: Devices that are used to display or produce results from the computer.
    
    eg: Display Screen, Speakers, Printer, etc
    

## CPU and Memory

CPU is brain of a computer. It performs all the computations.

Memory is used to store the data for processing and the processed data(result)

![[/image 45.png|image 45.png]]

**Types of memory:**

- Cache Memory:
    
    - It is places inside the CPU itself
    
    - Provides quick access to frequently used data.
    

- **Main Memory (RAM - Random Access Memory):**
    
    - Temporary storage for programs and data that are currently in use.
    
    - It is Volatile. Data in RAM is lost when the computer is turned off.
    
    - The programs are loaded to the Main memory from the Hard Disk (Secondary Memory) before execution
    

- **Secondary Storage (Hard Disk):**
    
    - Permanent storage for data and programs.
    
    - It is Non-Volatile. Data remains stored even when the computer is turned off.
    
    - eg: hard disks, DVDs, CDs, and USB drives.
    

### Execution of a program

Computer does not understand human language(English, Malayalam, etc). All it know is 0 and 1 (binary code).

![[/image 1 3.png|image 1 3.png]]

So we need some software that can translate human readable language to machine readable right! **Compilers** and **Interpreters** are used for it.

![[/image 2 3.png|image 2 3.png]]

- **Compiler**
    
    A compiler takes the entire code that you've written and converts it into machine language all at once, creating an executable file that the computer can run.
    

- **Interpreter**
    
    An interpreter converts and executes the code one line at a time, rather than processing the entire program at once.

**Platform Dependent** programs runs only on a particular machine (operating s/m or hardware specification)
